# Work Since

## Usage

```
npm i -g work-since
work-since YYYY-MM-DD
```

E.g:

```
$ work-since 2017-02-01

=> The date is over 4 years ago.
=>
=> In more detail:
=>  Distance in days: 1708 days in the past
=>  Distance in months: 56 months in the past
=>  Distance in years: 4.7 years in the past
```
